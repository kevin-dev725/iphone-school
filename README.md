# iPhone Photography School

This is the test project written by Kevin

## Languages and frameworks used

- Laravel 7.0
- PHP 7.2.5
- Vue 2.5

## Get started

### Run project

In root directly
``` bash
composer install

php artisan serve
```

Open a new terminal or duplicate current terminal tab.
``` bash
npm install
npm run watch
```

Go to localhost:8000
